# cth_mobile - Aplicación Móvil SientiaCTH

Aplicación móvil multiplataforma (Android/iOS) desarrollada con Flutter para el sistema de Control de Tiempo y Horarios (sientiaCTH).

## 🚀 Características
- **Fichaje Inteligente**: Entrada, salida y pausas desde el móvil.
- **Geolocalización**: Validación de ubicación en el fichaje.
- **NFC**: Soporte para etiquetas NFC en centros de trabajo.
- **Modo Offline**: Sincronización automática al recuperar conexión.
